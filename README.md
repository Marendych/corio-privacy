# Forma — Marketing Site

Static marketing site for [Forma](https://marendych.github.io/forma-privacy/), an iOS running and recovery tracking app.

## Pages
- **`index.html`** — Homepage with app overview, screenshots, and contact
- **`privacy.html`** — Privacy Policy (HealthKit, Oura, location, data handling)
- **`terms.html`** — Terms of Service (incl. medical disclaimer)

## Stack
Plain HTML + CSS, hosted on GitHub Pages from `main` branch.

## Local preview
```bash
python3 -m http.server 8080
# open http://localhost:8080/
```
